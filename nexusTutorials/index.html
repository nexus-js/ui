nexus

mobile audio interface elements

NexusUI is a JavaScript library for adding audio interface components to your website with HTML5 and transmitting their OSC data to other applications via AJAX.


Nexus can be used to integrate directly with Web Audio API in the browser, or to communicate with Max, SuperCollider or Chuck via server-side applications. It enables large-scale collaborative performances by distributing interfaces through a browser -- something that many people have at home, in their bag, or carry in their pocket. Nexus is mobile-friendly and multi-touch compatible.

Creating Elements
To create an interface with NexusUI, first link to jquery.js and nexusUI.js libraries

Any canvas with a valid nx property will be turned into a nexus element. Current nexus elements:
dial, button, toggle, slider, multislider, matrix, position, and keyboard
Accessing elements with JS
nx
Nexus is globally instantiated as nx

nx.onload
When Nexus loads, it triggers the nx.onload function which you can define in your javascript. Accessing Nexus elements should be done only after Nexus loads.

element names
By default, the first dial on the page will be given the js variable of dial1, etc. You can overwrite this default by giving the canvas an ID, which will become the element's js variable name.

Styling
size
Elements auto-adjust to fit the CSS or inline style of the canvas.

color
Nexus allows for global coloring with nx.colorize or setting individual element's properties. Parameters to set: accent (default), fill, border, accentborder, black, white.

drawing parameters
Some other drawing parameters can be set on a per object basis (forthcoming).
