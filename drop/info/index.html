DAISy

(Dynamic Audio Interface System)

A drag-and-drop environment for building and sharing web audio instruments

  • Try It!

DAISy is an extension of NexusUI and NexusDrop,
and features all NexusUI elements

Add elements to your interface by clicking and dragging

Write web audio functions for each individual element,
right in the browser.

This code will be executed each time the element is interacted with.

Save global web audio code to be executed once when your interface loads.

Save your interface to our interface database, and recall it later.
